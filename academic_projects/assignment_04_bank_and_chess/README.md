# Assignment 04 — Bank and Chess

This folder contains two Python exercises:

- `bank.py` — bank account and bank management
- `chess_tournament.py` — chess tournament management using dictionaries, recursion, and sorting

## Topics Used

- object-oriented programming
- classes and methods
- recursion
- sorting
- dictionaries and lists
- validation and error handling

## Run

Run the main file:

`python src/main.py`

Or run each project file separately:

- `python src/bank.py`
- `python src/chess_tournament.py`

## Tests

Run the tests with:

`python -m unittest discover -s tests`
