# Assignment 01 — Guessing Game and Knights Battle

This folder contains two Python exercises:

- `guessing_game.py` — a number guessing game with limited attempts and a hint after several guesses
- `knights_battle.py` — a simple battle simulation between two knights based on HP and attack power


## Topics Used

- variables and basic data types
- input and output
- conditional statements
- loops
- random number generation
- simple game logic

## Run

Run the main file:

`python src/main.py`

Or run each file separately:

- `python src/guessing_game.py`
- `python src/knights_battle.py`

## Tests

Run the tests with:

`python -m unittest discover -s tests`
