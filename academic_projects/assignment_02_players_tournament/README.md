# Assignment 02 — Players Tournament

This folder contains a Python tournament simulation project.

## Project Files

- `src/players_tournament.py` — player management, battle logic, and tournament simulation
- `src/main.py` — example entry point for running the project

## Topics Used

- functions
- lists and dictionaries
- loops
- conditional statements
- random number generation
- basic validation
- game simulation logic

## Run

Run the main file:

`python src/main.py`

Or run the project file directly:

`python src/players_tournament.py`

## Tests

Run the tests with:

`python -m unittest discover -s tests`
