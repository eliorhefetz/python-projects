# Assignment 05 — Hospital and Stack

This folder contains two Python exercises:

- `hospital_management.py` — hospital management using classes, queue, and priority queue
- `stack_exercises.py` — stack-based exercises

## Topics Used

- object-oriented programming
- classes and methods
- queue and priority queue
- stack
- recursion
- strings and lists

## Run

Run the main file:

`python src/main.py`

Or run each project file separately:

- `python src/hospital_management.py`
- `python src/stack_exercises.py`

## Tests

Run the tests with:

`python -m unittest discover -s tests`
