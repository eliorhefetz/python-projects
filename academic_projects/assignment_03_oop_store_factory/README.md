# Assignment 03 — OOP Store and Factory

This folder contains two object-oriented Python exercises:

- `store.py` — product, discount product, and store management
- `factory.py` — employee, manager, and factory management
- `main.py` — a simple menu for choosing which part to run

## Topics Used

- object-oriented programming
- classes and inheritance
- constructors and `__repr__`
- validation and error handling
- lists and tuples
- business logic methods

## Run

Run the main file:

`python src/main.py`

Or run each project file separately:

- `python src/store.py`
- `python src/factory.py`

## Tests

Run the tests with:

`python -m unittest discover -s tests`
